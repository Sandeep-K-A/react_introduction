# ShopEasy

A modern product listing application built with React, TypeScript, Tailwind CSS, and Vite.

## Features

- Product listing
- Search products
- Category filtering
- Product sorting
- Responsive UI
- Toast notifications

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Axios
- React Hot Toast
- Vite

## Setup

1. Clone the repository

2. Install dependencies

npm install

3. Create a .env file

cp .env.example .env

4. Start the development server

npm run dev

## Environment Variables

VITE_API_BASE_URL=https://fakestoreapi.com
